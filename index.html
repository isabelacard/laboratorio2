Opiniones personales

¿Que es el UX?

El diseño de experiencia de usuario (UX) se encarga de optimizar la interacción entre usuarios y productos, como sitios web y aplicaciones móviles. Inicialmente estudia los comportamientos y perspectivas de los usuarios para lograr que esta interacción sea amigable, entendible y útil. UX se refiere a la experiencia que tienen los usuarios cuando se benefician de un producto o servicio, y engloba todas las estrategias para optimizar la relación entre una marca y sus clientes.
